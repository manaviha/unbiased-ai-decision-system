# Unbiased AI Decision System

## 📌 Project Overview
The Unbiased AI Decision System is a web-based application developed using Django, Machine Learning, and Python. It analyzes uploaded datasets and detects bias in AI decision-making processes based on factors such as gender, income, age, and credit score.

## 🚀 Features
- Upload CSV dataset
- Machine learning model training
- Bias detection analysis
- Accuracy calculation
- Gender-wise approval comparison
- Result dashboard

## 🛠️ Technologies Used
- Python
- Django
- Pandas
- Scikit-learn
- HTML
- CSS
- SQLite
